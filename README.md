# martynazwoinska.github.io

The personal academic website of **Martyna Katarzyna Zwoińska**, evolutionary
biologist at Uppsala University. It is a single page of plain HTML and CSS
(`index.html`) — no build tools, nothing to install.

## Turning the website on (GitHub Pages)

The files are here, but GitHub still needs to be told to publish them:

1. On GitHub, open this repository → **Settings** (top menu) → **Pages** (left menu).
2. Under **Build and deployment → Source**, choose **Deploy from a branch**.
3. Set the branch to **`main`** and the folder to **`/ (root)`**, then **Save**.
4. Wait ~1 minute, refresh the Pages settings page, and it will show your live link.

### What your web address will be

Your GitHub username is `marzw`, and this repo is `martynazwoinska.github.io`, so
GitHub Pages will publish it at:

> **https://marzw.github.io/martynazwoinska.github.io/**

To get the shorter **https://martynazwoinska.github.io** instead, your GitHub
*username* would need to be `martynazwoinska` (Settings → Account →
Change username). That's free and quick — just say the word and I'll walk you
through it. Alternatively, a custom domain like `martynazwoinska.com` can be
pointed here later.

## Editing the site

Open `index.html` in any text editor. Everything is written in plain English
between the tags. Placeholders I left for you:

| What | Find this | Do this |
|------|-----------|---------|
| **Email** | `CHANGE-ME@ebc.uu.se` | Replace with your real email (appears twice) |
| **Photo** | `photo.jpg` (in the avatar block) | Add a photo — follow the comment right above it |
| **ORCID** | the `ORCID` link | Paste your ORCID URL |
| **Publications** | the Publications section | Add/remove entries |
| **Collaborators** | the Collaborators section | Add/remove people |

Or just tell me what to change and I'll edit and push it for you.
